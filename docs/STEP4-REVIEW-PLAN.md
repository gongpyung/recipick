```
- [ ] 제목 수정

- [ ] 재료 메모 수정
- [ ] 재료 추가 버튼 동작
- [ ] 재료 삭제 버튼 동작
- [ ] 조리 단계 텍스트 수정
- [ ] 조리 단계 추가 버튼 동작
- [ ] 조리 단계 삭제 버튼 동작
- [ ] 팁 수정/추가/삭제
```

### 2-3. 저장
```
- [ ] "저장하기" 버튼 → PATCH /api/recipes/:id 호출
- [ ] 저장 중 로딩 표시
- [ ] 저장 성공 → view 모드 전환 + 갱신된 데이터 표시
- [ ] 저장 실패 → 에러 메시지

```

### 2-4. 취소
```
- [ ] "취소" 버튼 존재

- [ ] dialog에서 "확인" → 편집 내용 폐기 + view 모드

```
